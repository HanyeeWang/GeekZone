# GeekZone
This is an information polymerized platform demo app for studying android.
# Main technique points
#### architecture : MVP + Dagger2 + Butterknife
#### network : Retrofit2.0 + OkHttp + Gson
#### imageloader : Glide
#### database : Realm
#### UI : Material Design
#### message : EventBus
#### performance : LeakCanary + Blockcanary
